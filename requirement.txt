Admin Login & Security
Login Form: A simple login page to access the system.
Route Protection: If a user is not logged in, they cannot view or modify any library data and must be redirected to the login page.
Secure Passwords: User passwords must be safely hashed in the database (never saved as plain text).
Validation
Prevent bad data: Stop users from entering negative stock quantities or leaving required fields blank.
Show a clear error message if validation fails.
